rsync server 的几个关键配置文件

  1. /etc/rsyncd.conf: 主配置文件;
  2. /etc/rsyncd.password/rsyncd.secrets: 秘钥文件;
  3. /etc/rsyncd.motd: rysnc 服务器元信息, 非必须;

其中, rsyncd.password 秘钥文件的掩码必须是 600:
